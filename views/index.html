Exercise tracker

Create a New User

POST /api/users

Add an Exercise:

POST

/api/users/username/exercise

GET user's exercise log: GET /api/users/:username/logs?[from][&to][&limit]

Example: GET /api/users/camillacab/logs?from=2022-01-01&to=2022-12-31&limit=2

[ ] = optional

from, to = dates (yyyy-mm-dd); limit = number
