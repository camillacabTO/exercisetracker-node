Exercise tracker

Create an Account:

POST /api/users

Get Exercises Log:

Send a GET request to: /api/users/:username/logs?[from][&to][&limit]

Example: GET /api/users/camillacab/logs?from=2022-01-01&to=2022-12-31&limit=2

[ ] = optional; from, to = dates (yyyy-mm-dd); limit = number

Enter an Exercise:

POTS /api/users/username/exercise
